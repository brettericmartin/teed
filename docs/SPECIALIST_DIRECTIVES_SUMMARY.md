## Research Methodology

### For All Specialists

**Interviews:**
- 30-60 minutes, semi-structured
- Record (with permission) and transcribe
- Document key quotes and insights

**Surveys:**
- 5-10 minutes, mobile-friendly
- Target 100+ responses minimum
- Include demographic questions

**Documentation:**
- Use consistent format
- Include executive summary
- Cite sources and data
- Include timelines
